## 프로젝트 소개

**스마트 차수판**은 어드벤처디자인 강의의 팀 프로젝트로 제작한 PWA(Progressive Web App) 웹 애플리케이션입니다. 홍수나 침수 상황에서 사용할 수 있는 스마트 차수판 시스템을 모바일 앱처럼 사용할 수 있도록 구현했습니다.

## 프로젝트 배경

홍수나 침수 상황에서 빠르게 대응할 수 있는 차수판 관리 시스템의 필요성을 느끼고, 누구나 쉽게 사용할 수 있는 모바일 웹앱을 개발했습니다. PWA 기술을 활용하여 앱스토어 설치 없이도 네이티브 앱처럼 사용할 수 있도록 구현했습니다.

## 주요 기능

- **차수판 상태 모니터링**: 실시간으로 차수판 설치 상태 확인
- **긴급 알림 시스템**: 침수 위험 시 즉각적인 알림
- **설치 가이드**: 차수판 설치 방법 안내
- **PWA 기능**: 홈 화면 추가, 오프라인 지원, 푸시 알림

## 기술 스택

- **Next.js**: 서버 사이드 렌더링 및 최적화
- **TypeScript**: 타입 안정성
- **PWA**: Progressive Web App 기술로 모바일 앱과 같은 사용자 경험 제공

## 개발 포인트

### PWA 첫 도전

처음으로 PWA를 구현해보며 Service Worker, Manifest 파일, 캐싱 전략 등을 학습했습니다. 특히 오프라인에서도 동작하는 웹앱을 만들면서 네트워크 상태에 따른 최적화를 고민할 수 있었습니다.

### 모바일 최적화

차수판 관리는 주로 현장에서 모바일로 사용될 것을 고려하여, 모바일 환경에 최적화된 UI/UX를 구현했습니다.

## 프로젝트 성과

강의 팀 프로젝트로 진행하며 기획부터 디자인, 개발, 발표까지 전 과정을 경험했습니다. 실제 문제를 해결하는 서비스를 만들면서 사용자 중심 개발의 중요성을 배웠습니다.
