## 프로젝트 소개

**이클래스 도와주는 코끼리(DGU-Ecko)**는 동국대학교 이클래스 LMS 시스템의 사용자 경험을 개선하는 크롬 익스텐션입니다. 학생들이 일상적으로 사용하는 이클래스의 불편한 점들을 찾아내고, 이를 해결하는 기능들을 추가했습니다.

## 해결하고자 한 문제

동국대학교 이클래스는 많은 학생들이 매일 사용하는 플랫폼이지만, 오래된 UI/UX와 불편한 기능들로 인해 사용성이 떨어진다는 피드백이 많았습니다. 이클래스 도와주는 코끼리는 이러한 문제들을 크롬 익스텐션을 통해 해결하고자 시작된 프로젝트입니다.

## 주요 기능

- **UI/UX 개선**: 더 직관적이고 현대적인 인터페이스 제공
- **편의 기능 추가**: 학생들의 학습 효율을 높이는 다양한 기능 구현
- **알림 기능**: 중요한 공지사항이나 과제 마감일 알림

## 기술 스택

- **React**: 익스텐션 UI 컴포넌트 구성
- **TypeScript**: 타입 안정성 및 코드 품질 향상
- **Chrome Extension API**: 브라우저 익스텐션 기능 활용

## 개발 포인트

크롬 익스텐션 개발이 처음이었지만, Chrome Extension API를 학습하며 웹 페이지에 스크립트를 주입하고 DOM을 조작하는 방법을 익힐 수 있었습니다. 또한 실제 사용자들의 피드백을 받으며 지속적으로 기능을 개선해나가는 과정에서 사용자 중심 개발의 중요성을 배웠습니다.
