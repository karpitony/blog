## 프로젝트 소개

**이클래스 도와주는 코끼리(DGU-Ecko)**는 동국대학교 이클래스 LMS 시스템을 더 편리하게 사용할 수 있도록 도와주는 크롬 확장 프로그램입니다. 학생들이 일상적으로 사용하는 이클래스의 불편한 점들을 개선하여 학습 관리를 더 쉽게 만듭니다.

## 주요 기능

### 학습 관리 개선

- **과제 마감일 한눈에 확인**: 모든 과제 마감일을 사이드패널에서 한눈에 파악
- **온라인 강의 시청 현황 확인**: VOD 시청 현황을 쉽게 확인
- **주차별 과제 목록 확장**: 기존 1주차에서 3주차로 확장하여 더 많은 과제를 한 번에 확인

### UX 개선

- **화면 가리는 공지 모달 숨기기**: 불필요하게 화면을 가리는 공지 모달 제거
- **사이드패널 UI**: React 기반의 직관적인 사이드패널로 정보 확인

## 기술 스택 및 아키텍처

### 기술 스택

- **WXT 프레임워크**: 크롬 익스텐션 개발 프레임워크
- **React**: 사이드패널 UI 컴포넌트 구성
- **TypeScript**: 타입 안정성 및 코드 품질 향상

### 레이어드 아키텍처 (Layered Architecture)

프로젝트는 명확한 책임 분리를 위해 레이어드 아키텍처 패턴을 적용했습니다:

- **Services**: 비즈니스 로직 (강의/VOD/과제 데이터 수집)
- **Managers**: 인프라 관리 (콘텐츠 스크립트, 탭, 메시지 관리)
- **Repositories**: 데이터 액세스 (Chrome Storage 추상화, 캐싱)

### 최적화 전략

- **캐시 우선 전략**: 불필요한 네트워크 요청 최소화 (강의ID: 1주일, VOD/과제: 4시간)
- **병렬 처리**: Promise.all로 여러 강의 동시 수집
- **콘텐츠 스크립트 중복 삽입 방지**: 마커 태그 활용으로 메모리 효율성 확보

## 개발 포인트

크롬 익스텐션 개발이 처음이었지만, **WXT 프레임워크**를 활용하며 Chrome Extension API를 학습했습니다. 특히 Background Script, Content Script, Side Panel 간의 메시지 통신 구조를 이해하고, 레이어드 아키텍처를 적용하여 유지보수하기 쉬운 코드를 작성하는 경험을 했습니다.

## 오픈소스 프로젝트

이 프로젝트는 **오픈소스**로 운영되며, 누구나 기여할 수 있습니다. 실제 사용자들의 피드백과 기여를 받으며 지속적으로 기능을 개선하고 있습니다.
